Introduction

Niwax - is a modern and creative HTML5 Multi-purpose template that is created for IT Companies, Marketing Agency, Startup, Freelancer Portfolio, Agency Portfolio, Design Studio, Marketing Landing Page, Web Design, Web Development, Graphic Design, App Development, Digital Marketing, Lead Generation, and any other online business solutions!. It comes with modern design and a light blazing fast loading template.

If you have any questions that are not addressed in this user manual, please contact us via the contact form on our profile page page. Contact Us

Template Help Guide

New Type Effect

1. Copy css from plugin.min.css at bottom to your exisitng website

2. Copy JS from plugin.min.js at bottom to your existing website

3. Copy Type Heading from Main index.html at top hero section and add to the your code.

4. Remove old js/typer.js from your page

How to Upgrade to Bootstrap v5

Replace Bootstrap Files from CSS and JS folder to your Code

Replace bootstrap.min.css

Replace bootstrap.bundle.min.js

After Replacing Following Files copy and paste css from style.css and responsive.css to your code. Note: Bootstrap 5 Css Patch is commented. Find it in style.css and responsive.css.

Some Plugins need manual changes in code. like Bootstrap Model, Tabs and According.

data-toggle="modal" to data-bs-toggle="modal"

data-toggle="tab" to data-bs-toggle="tab"

data-dismiss="modal" to data-bs-dismiss="modal"

Add -bs- in code to work in Bootstrap 5

How Configuration SMTP Form

Find the phpmailer/email/Mailer.php file from the email folder. Enter the SMTP information you received from your host here. See this link for more information

NOTE: On a local server, the contact form might not work. Upload and run the files on the live server.

Set Recipients Email: phpmailer/email/Mailer.php

How to Edit Email Template

Customize email template : phpmailer/email/send.php

NOTE: Contact form has two email templates. First for customers and second for admin. you can customize them in the send.php file according to your needs.
